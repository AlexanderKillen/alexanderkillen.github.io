Projects

Ultra-Low Latency C++20 HFT Matching Engine

A low-latency, deterministic limit order book delivering sub-35 ns median latency and >12M ops/sec throughput [Code].

The engine achieves O(1) performance by replacing conventional node-based structures with a bitmapped price-level approach. This design optimizes for L1 cache locality and hardware prefetching, effectively eliminating the pointer-chasing overhead found in traditional implementations.

Built on a zero-allocation architecture, the engine uses bit-packed order tokens for instant cancellations and an asynchronous, lock-free logging thread to isolate the critical matching path from I/O bottlenecks.

Self-designed Linear Audio Array

Built an 8-channel acoustic array [View Whole Project] [Code].

The system utilises an STM32 processing pipeline to sample sound from 8 PDM-mics, convert to PCM and package the data for transmission over USB to Python. Current capabilities include verification of synchronous sampling and an angle estimation algorithm (MUSIC) with real-time visualisation.
